# git-flash-cards-21c

## Git Workflow

Käytetään *Feature Branch* -workflowta.
Uutta ominaisuutta varten kukin luo itselleen feature branchin. Kun se on valmist, pushataan githubiin.

## Feature branchien nimeäminen

Käytetään etunimeä. Esim:
```feature-petteri-1```
